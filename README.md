# STN Workshop

Survive The Nights --- Modding & Workshop SDK

This repository contains the full Unity project used to build and run
the Survive The Nights Workshop system.

## Option 1 --- Git + LFS

``` bash
git lfs install
git clone https://github.com/Runelink2/STN-Workshop.git
cd STN-Workshop
git lfs pull
```
